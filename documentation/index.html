Getting Started

Follow these steps to quickly set up your coming soon page:

  1. Download and extract the template package
  2. Navigate to the script folder
  3. Choose your preferred template
  4. Copy the template files to your web server
  5. Configure the countdown timer and email subscription

Customization Guide

Countdown Timer

To modify the countdown target date, open script.js and update the following:

const targetDate = "2024-12-31";

Color Scheme

Customize colors by editing the CSS variables in style.css:

:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
}

Email Integration

To connect the subscription form to your email service:

  1. Open index.html
  2. Locate the form element
  3. Update the form action URL with your email service endpoint

Advanced Features

Custom Backgrounds

Each template supports different background customization:

  • Template 1: Image or gradient backgrounds
  • Template 3: Video background customization
  • Template 4 & 8: Particle system configuration
  • Template 6: SVG path customization
  • Template 7: Split-screen background images

Animation Settings

Adjust animation parameters in script.js:

// For particle templates
const particleConfig = {
  particles: {
    number: { value: 80 },
    speed: { value: 3 }
  }
};

// For morphing SVG
const morphDuration = 2000; // milliseconds

Troubleshooting

Common Issues

  • Countdown not working: Check if the target date is properly formatted
  • Email form not submitting: Verify the form action URL
  • Background media not loading: Ensure all file paths are correct
  • Animations not smooth: Check browser compatibility and reduce particle count

Browser Compatibility

The templates are tested and working on:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)

Performance Optimization

For better performance:

  • Optimize and compress images
  • Use appropriate video formats and sizes
  • Adjust particle count based on device capabilities
  • Minify CSS and JavaScript files
