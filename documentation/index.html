Getting Started

Follow these steps to quickly set up your coming soon page:

  1. Download and extract the template package
  2. Navigate to the script folder
  3. Choose your preferred template
  4. Copy the template files to your web server
  5. Configure the countdown timer and email subscription

Templates Overview

Template 1 - Classic Minimal

A clean, minimalist design perfect for professional websites.

Features:

  • Responsive design that works on all devices
  • Customizable countdown timer
  • Email subscription form with validation
  • Smooth background overlay effect
  • Easy color scheme customization

Customization Guide

Countdown Timer

To modify the countdown target date, open script.js and update the following:

const targetDate = "2024-12-31";

Color Scheme

Customize colors by editing the CSS variables in style.css:

:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
}

Email Integration

To connect the subscription form to your email service:

  1. Open index.html
  2. Locate the form element
  3. Update the form action URL with your email service endpoint

Advanced Features

Custom Backgrounds

You can use images, videos, or gradient backgrounds:

// For image background
.container .image {
  background: url('your-image.jpg');
}

// For video background
<video autoplay muted loop id="myVideo">
  <source src="your-video.mp4" type="video/mp4">
</video>

Troubleshooting

Common Issues

  • Countdown not working: Check if the target date is properly formatted
  • Email form not submitting: Verify the form action URL
  • Background image not showing: Ensure the image path is correct

Browser Compatibility

The templates are tested and working on:

  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
